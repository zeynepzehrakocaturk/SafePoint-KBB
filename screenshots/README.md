# Screenshots

Bu klasör, SafePoint KBB uygulamasının **gerçek ekran görüntülerini** içerir.

## Dosyalar

| Dosya | Açıklama |
|-------|----------|
| `dashboard.png` | Tam uygulama görünümü (masaüstü) |
| `map-view.png` | Leaflet harita ve marker'lar |
| `assembly-panel.png` | En yakın toplanma alanları paneli |
| `earthquake-panel.png` | Bugünkü depremler tablosu |
| `earthquake-modal.png` | Deprem detay modal penceresi |
| `mobile-view.png` | Mobil responsive görünüm |

## Yeniden Alma

```bash
npm start
# Yeni terminal:
npm run screenshots
```

Görseller Playwright ile Kocaeli koordinatları mock edilerek otomatik yakalanır.
